# 🐍 Snake Game (Python + Pygame)

A classic Snake Game built using Python and Pygame.

## 🎮 Features
- Smooth snake movement
- Food generation at random positions
- Collision detection (wall + self)
- Simple grid-based gameplay

## 🕹 Controls
- Arrow Up ⬆️
- Arrow Down ⬇️
- Arrow Left ⬅️
- Arrow Right ➡️

## 🚀 How to Run
```bash
pip install pygame
python snakegame.py
